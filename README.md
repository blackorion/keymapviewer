# Keymap Viewer

## install

run `yarn`

## develop

run `yarn start`

## maps
put a [name].json file into maps folder

config example:

```json
{
  "app": "app_name",
  "description": "",
  "keymap": [
    {
      "type": "shortcut",
      "description": "shortcut description",
      "keys": ["option", "j"]
    }
  ]
}
```
